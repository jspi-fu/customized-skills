# 思想汇报输出格式要求

## 标准结构
1. **标题**：居中写"思想汇报"
2. **称谓**：顶格写"敬爱的党组织："
3. **正文**（分3-4部分）：
   - a) 思想政治方面
   - b) 学习方面
   - c) 工作方面
   - d) 生活/纪律方面（可选）
4. **结尾**：请求和希望，如"恳请党组织给予批评、帮助"
5. **此致敬礼**："此致"空两格，"敬礼"顶格
6. **落款**：居右写"汇报人：×××"，下一行写"×年×月×日"

## 字数要求
| 章节 | 中文名称 | 字数限制 | 备注 |
|------|----------|----------|------|
| **总体报告** | **Total Draft** | **1700-1900字** | 3页，常规间距 |
| 引言（含称谓） | Introduction | ~100字 | 包含标题和称谓 |
| 思想政治方面 | Ideological/Political | 600-700字 | 核心内容，必须具有时效性 |
| 学习方面 | Study | 250-350字 | 可包含考试相关内容 |
| 工作方面 | Work | 250-350字 | 必须个性化 |
| 生活/纪律方面 | Life/Discipline | 100-150字 | 可选，主题：勤俭、团结、纪律 |
| 缺点与不足 | Self-Criticism | 150-200字 | 约2-3行 |
| 结尾与落款 | Conclusion | ~100字 | 标准结语短语和格式 |

## 示例输出格式

```
                              思想汇报

敬爱的党组织：

    自从被确定为入党积极分子以来，我始终以饱满的热情投入到理论学习和实
践锻炼中。本季度，我认真学习了……

    在思想政治方面，……

    在学习方面，……

    在工作方面，……

    在生活方面，……

    当然，我也清醒地认识到自身还存在一些不足。首先是……

    恳请党组织给予批评、帮助！

    此致
敬礼！

                                              汇报人：×××
                                              ××××年××月××日
```
