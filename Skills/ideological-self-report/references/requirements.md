# 思想汇报硬性要求

## 格式要求

### 页面设置
- 标题：居中写"思想汇报"
- 称谓：顶格写"敬爱的党组织："
- 正文：每段首行空两格
- 结尾："恳请党组织给予批评、帮助"
- 此致敬礼："此致"空两格，"敬礼"顶格
- 落款：居右写"汇报人：×××"，下一行写"×年×月×日"

### 字数要求
| 章节 | 字数限制 |
|------|----------|
| 总体报告 | 1700-1900字 |
| 思想政治方面 | 600-700字 |
| 学习方面 | 250-350字 |
| 工作方面 | 250-350字 |
| 生活/纪律方面 | 100-150字 |
| 缺点与不足 | 150-200字 |

## 结构要求

### 必须包含的章节
1. 思想政治方面
2. 学习方面
3. 工作方面
4. 缺点与不足

### 可选章节
- 生活/纪律方面（如字数已达标可省略）

## 内容要求

### 思想政治方面
- 结合最新时政热点
- 体现政治理论学习成果
- 联系自身实际谈认识

### 学习方面
- 体现专业知识学习
- 可包含考试、竞赛、论文等内容
- 突出学以致用

### 工作方面
- 具体工作职责和成果
- 服务意识和奉献精神
- 模范带头作用

### 缺点与不足
- 真实具体，不空泛
- 每季度不重复
- 体现改进决心

## 禁止事项

1. 禁止编造虚假信息
2. 禁止泄露国家秘密或敏感信息
3. 禁止抄袭他人内容
4. 禁止使用未经确认的政治表述
